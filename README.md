# Madcamp Week 2. MoneChat(먼챗)

## 프로젝트 소개

MoneChat은 사람들의 소비를 다시 한번 생각하게 해주는 거지방을 편하게 세팅해주는 앱입니다.

친구들을 초대해 함께 가계부를 쓰고 소비 내역을 공유하며 자신의 소비가 적절한지를 꼽을 받으며 재고해보세요!

당신의 소비는 확실히 줄어들 것입니다...

개발환경

FrontEnd:
  - Tool: JAVA
  - IDE: Android Studio

BackEnd:
  - Tool: Node.js
  - Server: AWS EC2
  - DB: MongoDB Atlas

Design: Figma
